On-Premise

A storage node is a machine with an operating system, CPU, memory, and file system. By installing NooBaa daemon, you actually turn this machine into NooBaa storage node.

Nodes will be assigned to nodes groups called “nodes pool” according to the administrator choice.

Cloud

A cloud resource can be either an Azure blob storage, AWS S3 bucket or any AWS S3 compatible service.

It can be used as a resource in the NooBaa buckets created by the administrator.
